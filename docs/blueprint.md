# **App Name**: Media Compass

## Core Features:

- Industry Overview: Provide a comprehensive introduction to the Entertainment and Media Communication major, detailing its curriculum, core concepts, and potential career paths.
- University Directory: Display a searchable and filterable list of Vietnamese universities offering programs related to Entertainment and Media Communication, with brief overviews.
- Admission Score Tracker: Present historical admission scores for each Vietnamese university and specific program from 2018 to the present, allowing users to analyze trends.
- Video Showcase: Embed relevant video content, such as program introductions, student testimonials, industry insights, and faculty interviews.
- Personalized Program Recommender: An AI tool that suggests relevant Vietnamese university programs or specialization tracks based on a user's interests, academic background, and career aspirations.
- Interactive Filters and Search: Enable users to efficiently search for specific universities, majors, or filter data by admission score ranges and other criteria.

## Style Guidelines:

- Primary color: A sophisticated, vibrant blue (#2259C3) to convey trustworthiness, innovation, and modern education, suitable for interactive elements and highlights.
- Background color: A very subtle, desaturated cool grey with a hint of blue (#F0F1F5), ensuring readability and a clean aesthetic.
- Accent color: A lively and distinct aqua (#47D0EA) to draw attention to call-to-actions, notifications, and key informational tags.
- Headlines: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and engaging visual appeal. Body text: 'Inter' (sans-serif) for clear readability across detailed information sections.
- Utilize minimalist, line-art style icons that are intuitive and convey educational themes, media elements, and data visualization concepts.
- Adopt a clean, grid-based layout with generous white space to enhance content hierarchy and legibility. Sections should be clearly delineated, facilitating easy navigation through university details and admission data.
- Incorporate subtle and smooth transition animations for filtering results, navigation between sections, and loading new content to create a polished and responsive user experience.
